01

Python Foundations

Build the base. No shortcuts. By end of this phase you'll write real Python programs and understand Git.

May – Jun 2026 · 6 weeks 2–3 hrs/day
Week 1 Python Setup + Variables + Data Types ▼
Day 1Mon
Install Python + VS Code. Watch "What is Python?" (30 min)
Print your name, age, city. Understand print() and strings
Day 2Tue
Variables: int, float, string, bool
Write 10 variable assignments. Do basic math operations in Python
Day 3Wed
Input from user (input()), type casting
Build: Simple calculator that adds two numbers entered by user
Day 4Thu
Strings: slicing, len(), upper(), lower(), f-strings
Build: A program that takes your name and prints a personalized greeting
Day 5Fri
Lists: create, index, append, remove, loop
Build: Shopping list program — add/remove/show items
Day 6Sat
Dictionaries: key-value pairs, get(), update()
Build: Student grade dictionary — store and retrieve grades
Day 7Sun
Review week 1. Redo any confusing exercises without looking at solutions
Write in a notebook: "What did I learn this week?" (5 things)
Resources
CS50P – Harvard (free) Python.org Tutorial W3Schools Python
Week 2 Control Flow + Loops + Functions ▼
Day 8Mon
if / elif / else — conditional logic
Build: Number guessing game (higher/lower hints)
Day 9Tue
for loops: iterating lists, range()
Print multiplication table, sum of 1–100
Day 10Wed
while loops + break + continue
Build: Login system with 3 attempts using while loop
Day 11Thu
Functions: def, parameters, return values
Write 5 functions: add, subtract, is_even, greet, celsius_to_fahrenheit
Day 12Fri
Scope, default arguments, *args
Build: BMI calculator using functions
Day 13Sat
Mini project: Build a "Contact Book" — add, search, delete contacts using lists + dicts + functions
Day 14Sun
Review + Solve 5 easy problems on HackerRank (Python track)
Week 3 Files + Error Handling + Modules ▼
Day 15Mon
Reading and writing files (open, read, write, with statement)
Save your contact book data to a .txt file
Day 16Tue
Try/except — handle errors gracefully
Add error handling to calculator (divide by zero, wrong input)
Day 17Wed
Importing modules: math, random, datetime
Build: Random quiz generator using random module
Day 18Thu
pip — installing external libraries. Install requests
Fetch a joke from a free API using requests library
Day 19Fri
List comprehensions + lambda functions (shortcuts)
Rewrite previous loops using comprehensions
Day 20Sat
Project: Weather CLI — fetch and display weather for any city using a free API
Day 21Sun
Rest + 5 more HackerRank problems
Week 4 Git + GitHub Basics ▼
Day 22Mon
What is Git? Why version control? Install Git
git init, git add, git commit on your contact book project

Create GitHub account. Push your first repo

02

Explore + Deepen Skills

Pick a direction. Try Data Science or Web. Start looking at open source orgs. Get AWS Cloud Practitioner certified.

Jul – Aug 2026 · 8 weeks 2–3 hrs/day
Week 7–8 Data Science Track (Try it out) ▼
Days 43–44
Install Jupyter Notebook. Learn NumPy basics — arrays, operations
Days 45–47
Pandas: load CSV, filter rows, group by, basic stats
Analyze a free dataset (Kaggle titanic or COVID data)
Days 48–49
Matplotlib: line charts, bar charts, scatter plots
Visualize your dataset — 3 different charts
Days 50–56
Mini project: Analyze COVID or population data for India. Create charts. Push notebook to GitHub
Optional: Try web dev with Flask (2 days) to see if you prefer it
Week 9–10 AWS Cloud Practitioner Prep ▼
Days 57–63
Study AWS CCP: EC2, S3, IAM, RDS, Lambda — 1 service/day
Use free tier: Launch your first EC2 instance. Host a static website on S3
Days 64–70
Practice 50 CCP exam questions daily (ExamTopics / Tutorials Dojo free)
Watch Stephane Maarek's CCP course (Udemy — buy on sale for ₹499)
Resources
AWS Skill Builder (free) ExamTopics CCP (free) Maarek CCP Course (₹499 on sale)
Week 11–14 Open Source Exploration + First Contributions ▼
Days 71–77
Browse gsoc.github.io — look at orgs from last 3 years. Shortlist 5 that interest you
Read their README, issues, and CONTRIBUTING.md files
Days 78–84
Pick 1 org. Set up their dev environment locally. Run the project
Filter GitHub issues by "good first issue" or "beginner friendly"
Days 85–98
Fix your first bug or improve docs. Open a Pull Request (even tiny ones count!)
Join their Discord/mailing list. Introduce yourself. Ask thoughtful questions
Apply for AWS Educate Student Ambassador (Q1 2027 — start building portfolio now)

Pick your GSoC org. Become a regular contributor. Build a stronger project. Get known in the community.

Sep – Nov 2026 · 12 weeks 2–3 hrs/day

Days 99–105
Advanced Python: decorators, generators, context managers, type hints
Rewrite one of your old projects using these concepts
Days 106–112
Unit testing with pytest — write tests for your own functions
Add tests to your weather CLI and contact book projects
Days 113–126
Target org: Solve 3–5 issues. Each PR should be non-trivial (not just typos)
Build a medium project (500+ lines): something related to your org's domain
Week 19–26 Consistent Contribution + Community ▼
Every week
Submit at least 1 PR per week to your target org
Participate in discussions, review others' PRs, answer questions in forums
Write a blog post about something you learned (dev.to or hashnode — free)
Oct–Nov
Reach out to 1–2 mentors in your org. Share what you've built. Ask for feedback
Organize a small AWS/cloud workshop at your college — key for AWS Ambassador app
Resources
dev.to (free blogging) Real Python (advanced topics) Your org's Discord/Mailing list

Phase 3 Complete — End of November 2026 10+ merged PRs. Known by at least 1 mentor. Blog with 5+ posts. College workshop done.

Refine proposal: be specific with week-by-week milestones and risk mitigation

Get proposal reviewed by mentor at least 2–3 times
Make a small proof-of-concept demo related to your proposal. Include link in proposal

Use this time to learn tools/tech you'll need for your project
